# Code of Conduct

## Expected Behavior

Contributors are expected to:

- communicate respectfully and focus criticism on the technical work
- assume good intent while still asking for concrete evidence and reproducible details
- keep discussions relevant to the repository
- avoid harassment, personal attacks, discriminatory language, and disclosure of private information

## Reporting

Report conduct concerns privately to the repository maintainers. Include links, screenshots, or message IDs when they
help establish context.

Maintainers may remove comments, close issues, restrict participation, or take other repository-level actions when a
participant repeatedly violates these expectations.

## Scope

This policy applies to repository issues, pull requests, discussions, code review, and project-linked communication
channels.
